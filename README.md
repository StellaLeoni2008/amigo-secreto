## 📌 Sobre
O *Sorteio de Amigo Secreto* foi desenvolvido como parte das atividades do curso **Alura – Lógica de Programação: praticando com desafios**.
O objetivo do projeto foi aplicar conceitos de lógica de programação utilizando JavaScript, criando uma aplicação simples que realiza o sorteio de amigo secreto entre os participantes informados pelo usuário.
Na aplicação, o usuário pode adicionar nomes a uma lista de participantes e, ao realizar o sorteio, o sistema embaralha os nomes e gera automaticamente as combinações de quem irá tirar quem no amigo secreto. O resultado é exibido diretamente na página, mostrando a relação entre os participantes sorteados.

## 🧠 Conceitos Fundamentais Praticados
Durante o desenvolvimento deste projeto, foram aplicados conceitos importantes de programação, como:
- Manipulação e armazenamento de dados em listas
- Adição dinâmica de elementos informados pelo usuário
- Atualização progressiva de conteúdo na interface
- Implementação de lógica de embaralhamento de dados
- Uso de operadores para percorrer e relacionar elementos de uma lista
- Geração de relações entre participantes no sorteio
- Reinicialização do estado da aplicação para permitir novos sorteios
- Organização da lógica do programa para execução passo a passo
