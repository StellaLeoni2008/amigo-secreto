## 📌 Sobre
O *Amigo Secreto* foi desenvolvido como parte das atividades do curso **Alura – Lógica de Programação: praticando com desafios**.

O objetivo do projeto foi praticar lógica de programação utilizando JavaScript para criar uma aplicação interativa de sorteio de amigo secreto.

A aplicação permite adicionar participantes a uma lista e realizar o sorteio de forma aleatória, trabalhando conceitos fundamentais de manipulação do DOM, listas e funções em JavaScript.

---

## ⚙️ Funcionalidades
- Adição de participantes
- Validação de nomes inseridos
- Exibição dinâmica da lista de participantes
- Sorteio aleatório do amigo secreto
- Atualização automática da interface

---

## 🧠 Conceitos Fundamentais Praticados
Durante o desenvolvimento deste projeto, foram aplicados conceitos essenciais de programação, como:
- Manipulação do DOM
- Arrays e listas
- Estruturas condicionais
- Funções
- Captura de dados inseridos pelo usuário
- Atualização dinâmica da interface
- Manipulação de eventos
- Geração de valores aleatórios com JavaScript
- Organização da lógica da aplicação
